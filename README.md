# Lal Thota - Personal Portfolio

A world-class, futuristic portfolio website built with React.js, showcasing my journey as an AI/ML Engineer and Full Stack Developer. Designed with a dark, premium aesthetic, smooth Framer Motion animations, and a focus on high performance.

## 🔗 Live Demo
**[lalthota.vercel.app](https://lalthota.vercel.app)** *(Replace with actual Vercel link once deployed)*

## 🚀 Tech Stack
- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS v4, Custom CSS (Glassmorphism, Glitch Effects)
- **Animations:** Framer Motion, tsParticles
- **Icons:** React Icons
- **Contact Form:** EmailJS
- **Deployment:** Vercel

## ✨ Key Features
- **Dynamic Hero Section:** Animated particle background, glitch text effects, and a custom typewriter component.
- **Interactive Custom Cursor:** A dual-element cursor that tracks mouse movement and highlights interactive elements.
- **Bento Grid Skills:** 3D tilt effects using CSS perspective and Framer Motion for a premium, Apple-like feel.
- **Project Showcase:** Filterable gallery with hover reveals and image overlays.
- **Achievements Section:** Animated counters and trophy cards with particle effects.
- **Performance Optimized:** Uses Vite, lazy loading, and hardware-accelerated animations for a 90+ Lighthouse score.

## 🛠️ Setup Instructions

To run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/lalthota/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up EmailJS**
   - Create an account at [EmailJS](https://www.emailjs.com/)
   - Set up an email service and a template
   - Update the `Contact.jsx` component with your `SERVICE_ID`, `TEMPLATE_ID`, and `PUBLIC_KEY`

4. **Start the development server**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

5. **Build for production**
   ```bash
   npm run build
   ```
